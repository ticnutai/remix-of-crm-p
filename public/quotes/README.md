# מערכת הצעות מחיר - MALI TENENBAUM

## סקירה כללית
מערכת מתקדמת לניהול וייצוא הצעות מחיר עבור שירותי עיצוב פנים.

---

## 🎨 עיצוב ותכונות

### ערכות צבעים זמינות
- **זהב (Gold)** - ערכת ברירת המחדל עם גוונים חמים וזהובים
- **כחול (Blue)** - מראה מקצועי ומודרני
- **ירוק (Green)** - טבעי ורגוע
- **סגול (Purple)** - יוקרתי ואלגנטי

### תבניות ייצוא
- **קלאסי** - עיצוב מסורתי עם גבולות ברורים וגרדיאנט זהב
- **מודרני** - עיצוב עכשווי עם צבעים נועזים וגרדיאנט סגול
- **מינימליסטי** - עיצוב נקי ופשוט בשחור-לבן

### תכונות מתקדמות
- ✅ מצב כהה (Dark Mode)
- ✅ אפקטי Glassmorphism
- ✅ גרירה ושחרור לסידור מחדש
- ✅ תצוגת רשת/רשימה
- ✅ סינון וחיפוש מתקדם
- ✅ גרפים ותרשימים (עוגה + Gantt)
- ✅ אפקט קונפטי לאישור הצעות
- ✅ מערכת תיקיות לארגון

## 🛠️ טכנולוגיות

### Frontend
- **React 18** - ספריית UI
- **TypeScript** - שפת תכנות עם טיפוסים
- **Tailwind CSS** - עיצוב מודולרי
- **Framer Motion** - אנימציות מתקדמות

### ספריות עזר
- **@dnd-kit** - גרירה ושחרור
- **recharts** - גרפים ותרשימים
- **jsPDF + html2canvas** - ייצוא PDF
- **docx** - ייצוא Word
- **JSZip** - ייצוא ZIP

## 📋 מבנה הצעת מחיר

כל הצעת מחיר מכילה:

1. **כותרת וסיכום**
   - שם ההצעה
   - תת-כותרת
   - מיקום
   - מחיר כולל

2. **חבילות מחיר** (אופציונלי)
   - שם החבילה
   - מחיר
   - תיאור
   - רשימת תכונות
   - סימון "מומלץ"

3. **שידרוגים ותוספות** (אופציונלי)
   - שם השידרוג
   - מחיר
   - תיאור

4. **סעיפים ופריטים**
   - כותרת סעיף
   - רשימת פריטים
   - תוספת מחיר לפריטים (אופציונלי)

5. **לוח תשלומים**
   - אחוז תשלום
   - תיאור/עיתוי

6. **לוח זמנים** (אופציונלי)
   - שלבי הפרויקט

7. **הערות חשובות**
   - תנאים והגבלות
