# בקרות Git בכלי פיתוח צפים 🔧

## תיאור
כפתורים צפים לניהול Git ישירות מתוך הדפדפן במצב פיתוח.

## תכונות

### 🔵 Git Pull - משיכת שינויים
- לחיצה על הכפתור הכחול מעתיקה את הפקודה `git pull` ללוח
- פשוט הדבק בטרמינל והרץ
- צבע כחול עם אייקון GitPullRequest

### 🟢 Git Push - דחיפת שינויים
- לחיצה על הכפתור הירוק מעתיקה פקודה מלאה:
  ```bash
  git add . && git commit -m "Auto commit from DevTools" && git push
  ```
- מבצע add, commit ו-push בפעולה אחת
- צבע ירוק עם אייקון Upload

## איך להשתמש

### 1. הפעלת כלי Git
1. עבור להגדרות (Settings)
2. לחץ על "הגדרות מפתחים"
3. הפעל את "מצב פיתוח"
4. הפעל את "בקרות Git" (מופעל כברירת מחדל)

### 2. משיכת שינויים מ-GitHub
1. לחץ על הכפתור הכחול (Git Pull) בפינה השמאלית התחתונה
2. יופיע הודעה: "פקודת Git Pull הועתקה ללוח"
3. פתח טרמינל
4. הדבק (Ctrl+V או Cmd+V)
5. הרץ את הפקודה

### 3. דחיפת שינויים ל-GitHub
1. לחץ על הכפתור הירוק (Git Push) בפינה השמאלית התחתונה
2. יופיע הודעה: "פקודת Git Push הועתקה ללוח"
3. פתח טרמינל
4. הדבק (Ctrl+V או Cmd+V)
5. הרץ את הפקודה - תבצע commit + push אוטומטי

## מיקום הכפתורים
הכפתורים מופיעים בכלי הפיתוח הצפים בצד שמאל למטה של המסך, לצד:
- 🟡 קונסול מפתחים
- 🟡 זיהוי אלמנטים (Inspector)
- 🟡 מד ביצועים
- 🗑️ ניקוי קאש

## לוגים
כל פעולת Git נרשמת בקונסול הפיתוח עם:
- 📋 אייקון ללוח העתקה
- ⏰ חותמת זמן
- 📝 תיאור הפעולה

## הגדרות נוספות

### כיבוי/הפעלה של כפתורי Git
בהגדרות מפתחים, תחת "שליטה בכלים בודדים":
- ✅ מופעל - הכפתורים מופיעים
- ❌ כבוי - הכפתורים מוסתרים

### העתקה ידנית של פקודות
אם תרצה להעתיק פקודות Git ספציפיות:

**Git Pull:**
```bash
git pull
```

**Git Push עם Commit:**
```bash
git add . && git commit -m "Auto commit from DevTools" && git push
```

**Git Status:**
```bash
git status
```

## טיפים

### 💡 טיפ 1: בדיקת סטטוס לפני Push
לפני שתבצע Push, מומלץ לבדוק מה משתנה:
```bash
git status
git diff
```

### 💡 טיפ 2: הודעת Commit מותאמת אישית
אם תרצה הודעת commit מפורטת יותר:
```bash
git add .
git commit -m "תיאור השינויים שלך כאן"
git push
```

### 💡 טיפ 3: שימוש עם Branches
אם אתה עובד על branch מסוים:
```bash
git pull origin your-branch-name
git push origin your-branch-name
```

## פתרון בעיות

### הכפתורים לא מופיעים
1. ודא שמצב פיתוח מופעל
2. בדוק שבקרות Git מופעלות בהגדרות
3. רענן את הדף (F5)

### הפקודה לא מועתקת
1. בדוק הרשאות clipboard בדפדפן
2. נסה לאפשר הרשאות clipboard עבור האתר

### הפקודה נכשלת בטרמינל
1. ודא שאתה בתיקיית הפרויקט הנכונה
2. בדוק שיש לך הרשאות Git
3. ודא ש-Git מותקן במערכת

## קיצורי מקלדת מומלצים

בטרמינל:
- `Ctrl + V` / `Cmd + V` - הדבק פקודה
- `Enter` - הרץ פקודה
- `Ctrl + C` - בטל פקודה רצה
