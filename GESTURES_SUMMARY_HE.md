# סיכום: מערכת הגדרות מחוות סיידבר

## מה נוסף?

הוספתי מערכת **הגדרות מחוות מתקדמת** לסיידבר שמאפשרת לך להתאים אישית את כל ההתנהגויות והמחוות.

## איך לגשת?

1. פתח את הסיידבר (רחף על הקצה הימני של המסך)
2. גלול לתחתית
3. לחץ על כפתור **"יד" 🤚** ליד כפתור הצבעים
4. תיפתח חלונית הגדרות מתקדמת

## מה אפשר להגדיר?

### 1️⃣ הסתרה אוטומטית
- **הפעל/כבה** את ההסתרה האוטומטית
- **זמן המתנה**: 100-2000 מילישניות (ברירת מחדל: 800ms)
- ככל שהזמן גבוה יותר - הסיידבר נשאר פתוח יותר זמן

### 2️⃣ פתיחה בריחוף
- **הפעל/כבה** פתיחה אוטומטית כשהעכבר מתקרב
- **רגישות**: 10-100 פיקסלים (ברירת מחדל: 50px)
- רגישות גבוהה = נפתח מרחק רב יותר מהקצה

### 3️⃣ הצמדה (Pin)
- **הפעל/כבה** יכולת הצמדה
- **זכור מצב**: שמור את מצב ההצמדה גם לאחר רענון

### 4️⃣ שינוי גודל
- **הפעל/כבה** יכולת שינוי רוחב
- **רוחב מינימלי**: 150-300px (ברירת מחדל: 200px)
- **רוחב מקסימלי**: 300-600px (ברירת מחדל: 400px)

### 5️⃣ טריגר קצה
- **הפעל/כבה** אזור הפעלה בקצה המסך
- **רוחב אזור**: 3-50 פיקסלים (ברירת מחדל: 12px)
- אזור רחב יותר = קל יותר להפעיל

### 6️⃣ החלקה במובייל
- **הפעל/כבה** מחוות החלקה במכשירים ניידים
- **סף החלקה**: 20-150 פיקסלים (ברירת מחדל: 50px)

### 7️⃣ מהירות אנימציה
- **מהירות**: 100-1000 מילישניות (ברירת מחדל: 300ms)
- מהיר = תנועות חדות, איטי = תנועות חלקות

## תכונות נוספות

### 💾 שמירה אוטומטית
כל השינויים נשמרים ב-localStorage ונזכרים גם לאחר סגירת הדפדפן.

### 🔄 איפוס להגדרות ברירת מחדל
כפתור "אפס להגדרות ברירת מחדל" מחזיר את כל ההגדרות למצב המומלץ.

### 🎨 משולב עם הגדרות עיצוב
ההגדרות החדשות עובדות יחד עם הגדרות העיצוב הקיימות (צבעים, רוחב, גופן).

### ⚡ ביצועים
כל ההגדרות משפיעות בזמן אמת - אין צורך לרענן את הדף.

## איך זה עובד טכנית?

```typescript
// מבנה ההגדרות
interface SidebarGesturesConfig {
  autoHideEnabled: boolean;
  autoHideDelay: number;
  hoverEnabled: boolean;
  hoverSensitivity: number;
  pinEnabled: boolean;
  pinRememberState: boolean;
  resizeEnabled: boolean;
  minWidth: number;
  maxWidth: number;
  edgeTriggerEnabled: boolean;
  edgeTriggerWidth: number;
  swipeEnabled: boolean;
  swipeThreshold: number;
  animationSpeed: number;
}
```

ההגדרות נטענות מ-localStorage בעת אתחול הקומפוננט:
```typescript
const [gesturesConfig, setGesturesConfig] = useState<SidebarGesturesConfig>(
  () => loadGesturesConfig()
);
```

וכל שינוי מיידית משפיע על התנהגות הסיידבר.
